# Analyse de l'inflation au Canada avec des modèles ARMA

## Description du projet

Ce projet analyse l'évolution de l'inflation au Canada en utilisant deux indicateurs :

- IPC Global
- IPC HAEI (hors alimentation, énergie et effets des impôts indirects)

Les données couvrent la période 1995–2024 avec une fréquence mensuelle.

L'objectif est d'étudier la dynamique des séries temporelles et d'identifier des modèles ARMA adaptés.

---

## Méthodologie

L'analyse comprend les étapes suivantes :

- exploration des séries temporelles
- statistiques descriptives
- filtrage (lissage linéaire et exponentiel)
- analyse de stationnarité
- différenciation
- analyse ACF et PACF
- estimation de modèles ARMA
- diagnostics des modèles
- analyse des corrélations croisées

---

## Résultats principaux

Les modèles retenus sont :

- MA(1) pour la série IPC-HAEI
- MA(2) pour la série IPC-GLOBAL
